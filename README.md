A aplicação web permite registar interações do utilizador através de quatro botões.
Sempre que um botão é clicado, o sistema regista a data, a hora e um número sequencial diário, que reinicia automaticamente a cada novo dia.
Os dados são guardados numa base de dados SQLite, garantindo que o registo se mantém mesmo após atualizar a página.
O projeto foi desenvolvido com HTML, CSS e JavaScript no front-end, Python com Flask no back-end, versionado no GitHub e executado no Replit através de um link público.
